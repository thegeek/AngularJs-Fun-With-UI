AngularJs Fun With UI Codepen

This is my first fork of AngularJs Template ( here )

Each Controller has his own template.

In this example there are 3 type of controller:

  • The main controller that aim to inform the user about the experiment throught a simple html view. Here link in the controller explanation have a popover-ui feature.
  • Todo List Controller ( here ) show a simple Todo List, you can add, remove and checks Todos. Removal has a confirm modal-ui dialog.
  • Contact List Controller ( here ) show another simple controller. You can search by all field of the data or just for specific fields (name, phone, email). Search box has typeahead-ui feature, and suggest are shown for type (if you are searching for phone it'll suggest the phone numbers that match with the search value
